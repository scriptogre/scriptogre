#### Hyperspace ([try it out](https://hyperspace.christiantanul.com) / [source code](https://github.com/scriptogre/hyperspace))

I built another multiplayer game (?) as an experiment trying to find the limits of real-time hypermedia (using htmx 😎).

<img width="500" alt="hyperspace" src="https://github.com/user-attachments/assets/9fc715ce-9573-4993-972c-bf4da67aa10d" />

It uses a [patched version](https://github.com/scriptogre/SpacetimeDB/tree/hypermedia) of [SpacetimeDB](https://spacetimedb.com/) that enables streaming HTML directly from the database. 

Every user action is synced to the DB & streamed to connected users. The entire "stack" is SpacetimeDB + a .jinja2 template. 

It might feel laggy until I add client-side prediction.
